# 3D Wave Grid

This project is an interactive, high-performance 3D experience built with **Three.js** and custom **GLSL Shaders**, featuring an interactive 40x40 cubes grid that responds to mouse movement with fluid, ripple-like wave simulations.

## Tech Stack

- **Three.js** - 3D Engine
- **GLSL** - Custom Shaders
- **GSAP** - Animation Library
- **Vite** - Build Tool
- **lil-gui** - Debug UI

## Getting Started

### Prerequisites

Detailed setup requires **Node.js** and **npm** (or **pnpm**).

### Development

You can use **npm** or **pnpm** to manage dependencies:

```bash
# Install dependencies
pnpm install

# Start local server
pnpm dev

# Build for production
pnpm build
```
